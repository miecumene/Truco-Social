# Truco Social 3D

Proyecto: Truco Social 3D — juego multijugador social basado en Truco en un entorno 3D.

Estado: WIP  
Documento de diseño (GDD): `docs/GDD.md`

## Estructura del repositorio
- docs/ — Documentación y diseño del juego (GDD, mecánicas, UI/UX, networking).
- unity/TrucoSocial3D/ — Proyecto Unity (no incluir `Library/`, `Temp/`, `Build/`).
- assets/ — Concept art y referencias.
- .github/ — Plantillas de issues y workflows (CI opcional).

## Requisitos
- Unity (versión indicada en `unity/TrucoSocial3D/ProjectSettings/ProjectVersion.txt`)
- Git
- Git LFS (recomendado para assets grandes)

## Cómo empezar (local)
1. Clona el repo:
   ```bash
   git clone git@github.com:miecumene/Truco-Social.git
   cd Truco-Social
   ```
2. Instala Git LFS y registra los tipos de archivo grandes:
   ```bash
   git lfs install
   ```
3. Abre el proyecto en Unity desde `unity/TrucoSocial3D`.
